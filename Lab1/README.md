# Лабораторная работа №1

## Тема

Базовое знакомство с Java: методы, циклы, условные операторы, работа со строками и запуск программ через JVM.

## Состав

| Файл | Назначение |
|---|---|
| `Primes.java` | Вывод простых чисел от 2 до 100 |
| `Palindrome.java` | Проверка строк на палиндром |

## Запуск

```bash
javac *.java
java Primes
java Palindrome madam racecar apple kayak song noon
```
