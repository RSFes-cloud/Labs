# Лабораторная работа №2

## Тема

Объектно-ориентированное программирование в Java: классы, абстрактные классы, наследование, перегрузка методов, переопределение методов и инкапсуляция.

## Состав

| Файл | Назначение |
|---|---|
| `Appliance.java` | Базовый абстрактный класс бытового прибора |
| `KitchenAppliance.java` | Расширение базового класса для кухонной техники |
| `Refrigerator.java` | Класс холодильника |
| `Dishwasher.java` | Класс посудомоечной машины |
| `VacuumCleaner.java` | Класс пылесоса |
| `Main.java` | Демонстрация работы иерархии классов |

## Запуск

```bash
javac *.java
java Main
```

При запуске программа запрашивает бренд холодильника и температуру. Значения вводятся с клавиатуры.
