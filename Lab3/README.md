# Лабораторная работа №3

## Тема

Хэш-таблицы и хранение данных по принципу «ключ-значение». В работе есть собственная реализация хэш-таблицы с обработкой коллизий, а также демонстрация работы со стандартным `HashMap` на примере базы студентов.

## Состав

| Файл | Назначение |
|---|---|
| `HashTable.java` | Реализация хэш-таблицы |
| `Main.java` | Демонстрация добавления, поиска и удаления студентов через `HashMap` |

## Запуск

```bash
javac *.java
java Main
```
