# Лабораторная работа №4

## Тема

Обработка исключений и работа с файлами в Java: стандартные исключения, пользовательское исключение, блок `finally`, логирование ошибок и копирование файлов.

## Состав

| Файл | Назначение |
|---|---|
| `ArrayAverage.java` | Расчет среднего значения массива с обработкой ошибок |
| `DivisionLogger.java` | Деление чисел, собственное исключение и запись ошибок в лог |
| `FileCopier.java` | Копирование файла с обработкой ошибок ввода-вывода |

## Запуск

```bash
javac *.java
java ArrayAverage
java DivisionLogger
java FileCopier
```

Для `FileCopier.java` нужен файл `source.txt` в папке лабораторной работы. Результат копирования будет записан в `destination.txt`.
