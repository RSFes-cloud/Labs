# Лабораторная работа №5

## Тема

Регулярные выражения в Java. Работа выполняется через классы `Pattern` и `Matcher`.

## Состав

| Файл | Назначение |
|---|---|
| `NumberFinder.java` | Поиск чисел в строке |
| `PasswordValidator.java` | Проверка корректности пароля |
| `CapitalLetterHighlight.java` | Поиск заглавной буквы после строчной и выделение найденных случаев |
| `IPValidator.java` | Проверка корректности IP-адреса |
| `WordFinder.java` | Поиск слов, начинающихся с заданной буквы |

## Запуск

```bash
javac *.java
java NumberFinder
java PasswordValidator
java CapitalLetterHighlight
java IPValidator
java WordFinder
```
