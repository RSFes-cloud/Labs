# Лабораторная работа №6

## Тема

Коллекции Java и обобщения: `ArrayList`, `HashMap`, сортировка записей коллекции, generic-класс и учет данных в прикладной задаче.

## Состав

| Файл | Назначение |
|---|---|
| `TopWords.java` | Подсчет наиболее часто встречающихся слов в текстовом файле |
| `text.txt` | Исходный текст для `TopWords.java` |
| `Stack.java` | Собственная обобщенная реализация стека |
| `SalesTracker.java` | Учет проданных товаров и подсчет общей суммы продаж |

## Запуск

```bash
javac *.java
java TopWords
java Stack
java SalesTracker
```

`TopWords.java` читает файл `text.txt`, поэтому запускать программу нужно из папки `Lab6`.
