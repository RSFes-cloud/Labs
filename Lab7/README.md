# Лабораторная работа №7

## Тема

Многопоточное программирование в Java: создание потоков, ожидание завершения через `join()`, разделение вычислений между потоками и синхронизация доступа к общему ресурсу.

## Состав

| Файл | Назначение |
|---|---|
| `ArraySumThread.java` | Расчет суммы элементов массива двумя потоками |
| `MatrixMaxThread.java` | Поиск максимального элемента матрицы с обработкой строк в отдельных потоках |
| `WarehouseTransfer.java` | Модель переноса товаров со склада несколькими грузчиками |

## Запуск

```bash
javac *.java
java ArraySumThread
java MatrixMaxThread
java WarehouseTransfer
```
