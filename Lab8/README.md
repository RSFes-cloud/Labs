# Лабораторная работа №8

## Тема

Аннотации, Reflection, Stream API и многопоточная обработка данных через `java.util.concurrent`.

## Состав

| Файл | Назначение |
|---|---|
| `Main.java` | Основная программа: аннотация `@DataProcessor`, обработчики данных, `DataManager` и тестовый запуск |
| `Lab8_explanation.md` | Подробное объяснение лабораторной работы |
| `Lab8_questions.md` | Вопросы и ответы для защиты |

## Что делает программа

Программа создает тестовый файл `input.txt`, загружает строки, применяет к ним обработчики, помеченные аннотацией `@DataProcessor`, выполняет обработку в пуле потоков и сохраняет результат в `output.txt`.

## Запуск

```bash
javac Main.java
java Main
```

После запуска в папке появятся файлы `input.txt` и `output.txt`.
