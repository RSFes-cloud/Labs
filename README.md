# Лабораторные работы по Java

Учебный репозиторий с лабораторными работами по дисциплине, связанной с кроссплатформенными технологиями программирования и основами Java. Внутри находятся исходные файлы `.java`, краткие описания заданий и команды для запуска.

## Содержание

| № | Папка | Тема |
|---|---|---|
| 1 | [Lab1](Lab1) | Базовый синтаксис Java, методы, циклы, строки, простые числа и палиндромы |
| 2 | [Lab2](Lab2) | Объектно-ориентированное программирование: классы, наследование, инкапсуляция и полиморфизм |
| 3 | [Lab3](Lab3) | Хэш-таблица: хранение пар «ключ-значение», обработка коллизий, работа с собственной структурой данных |
| 4 | [Lab4](Lab4) | Исключения и работа с файлами: `try-catch-finally`, собственное исключение, логирование и копирование файлов |
| 5 | [Lab5](Lab5) | Регулярные выражения: поиск чисел, проверка пароля и IP-адреса, обработка текста |
| 6 | [Lab6](Lab6) | Коллекции и обобщения: `Map`, `List`, собственный generic-стек, учет продаж и подсчет слов |

| 8 | [Lab8](Lab8) | Аннотации, Reflection, Stream API и `java.util.concurrent` для обработки данных |

## Требования

Для запуска нужен установленный JDK. Проверить установку можно командой:

```bash
java -version
javac -version
```

## Общий способ запуска

Перейдите в папку нужной лабораторной работы, скомпилируйте исходники и запустите нужный класс:

```bash
cd Lab1
javac *.java
java Primes
```

Для лабораторных, где несколько независимых программ, запуск выполняется по имени соответствующего класса, например `java Palindrome`, `java PasswordValidator`, `java MatrixMaxThread`.
